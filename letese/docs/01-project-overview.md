# Letese.xyz — Project Overview

## What is Letese?
**Letese** is a Legal Tech SaaS platform that simplifies legal and professional services for businesses and individuals. It provides end-to-end compliance management, expert guidance, and seamless procedural support.

## Mission
> "Simplify legal services with technology — making compliance accessible, transparent, and hassle-free."

## Core Services
- **Company Registration** — Private Ltd, Public Ltd, OPC, LLP, Foreign Entities
- **Fire NOC** — Fire Safety NoC compliance and guidance
- **CA Services** — Chartered Accountant consultation and filing
- **CS Services** — Company Secretary compliance support
- **Legal Advisory** — Expert legal consultation

## Tech Stack

| Layer | Technology |
|---|---|
| **Frontend** | FlutterFlow (No-code/Low-code) |
| **Backend** | FastAPI (Python) |
| **Database** | PostgreSQL |
| **Auth** | JWT + Email OTP |
| **File Storage** | AWS S3 / Cloudflare R2 |
| **Email** | SendGrid / Resend |
| **Hosting** | Vercel (FlutterFlow) + Railway/Render (Backend) |

## Target Users
- **Businesses** — Startups, SMEs needing company registration & compliance
- **Property Managers** — Fire NOC and safety compliance
- **Individuals** — Personal legal advisory needs
- **Professionals** — CAs, CSs, Lawyers using the platform

## Version
- **v0.1** — MVP: Auth, User Dashboard, Service Catalog, Case/Service Request Creation
- **v0.2** — Document Upload, Payment Integration, Case Tracking
- **v1.0** — Full Platform: Payments, Notifications, Admin Panel, Analytics
