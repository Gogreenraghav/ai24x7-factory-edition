## 1. PRODUCT OVERVIEW

### What is AI24x7 Vision?

AI24x7 Vision is a CCTV AI analytics system that runs 100% OFFLINE on customer premises.
No cloud dependency. No internet required for AI analysis.

**How it works:**
1. Customer buys a hardware machine (we configure)
2. AI24x7 agent + Fine-tuned model installed
3. CCTV cameras connect to machine via RTSP
4. Machine analyzes frames locally (llama.cpp + Fine-tuned v10)
5. Suspicious activity detected → Telegram alert sent to customer

**Key Features:**
- 24/7 CCTV monitoring
- AI-powered detection (person, vehicle, unknown person)
- Hindi/English Telegram alerts
- Real-time dashboard (local)
- Auto daily reports (Google Sheets)
- Voice AI option available (Vision+Voice)

## 7. INSTALLATION PROCESS

### Customer Machine Setup (1 Command):

```bash
curl -sSL https://raw.githubusercontent.com/Gogreenraghav/ai24x7-vision/main/cctv-ai/install.sh | bash
```

This automatically:
1. Installs Python + dependencies
2. Downloads AI24x7 Fine-tuned v10 model
3. Configures llama.cpp
4. Sets up systemd service
5. Configures Telegram bot
6. Creates startup script

### Steps After Install:
1. Edit `camera_config.py` - Add RTSP URLs
2. Restart service
3. Test with sample CCTV feed
4. Configure Telegram Chat ID
5. Set up daily reports (optional)

### System Requirements for Customer Machine:
- Ubuntu 20.04 / 22.04 LTS
- 16GB RAM minimum
- 500GB SSD minimum
- NVIDIA GPU (RTX 3060 8GB minimum)
- Internet for Telegram alerts
